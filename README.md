# P5 スリースターズ練習

FF14「絶妖星乱舞」P5のスリースターズを練習するための非公式練習ページです。

9本塔、属性耐性低下、無職、1回目・3回目の2択カタストロフィを見ながら、自分がどの塔のどの位置に入るかを確認できます。

## できること

- スリースターズの位置確認
- スリースターズの実践練習
- MT / ST / H1 / H2 / D1 / D2 / D3 / D4 の自分ロール選択
- パーティーリスト形式でのデバフ確認
- 氷 / 雷 / 炎 の耐性低下と無職の割り振り
- 塔を踏んだ後に追加で付く耐性低下デバフの表示
- 複数デバフの表示順を `雷 → 炎 → 氷` に固定
- 1回目・3回目のカタストロフィ中外判断
- 実践練習での制限時間バー、デバフ残り秒数表示
- マウスクリック、WASD、矢印キー、ゲームパッドでの移動
- 実践中・一時停止中どちらでもスライド送り
- マーカー配置 `2341 / 1234` の切り替え

## 基本ルール

スリースターズでは、氷・雷・炎の塔が3本ずつ、合計9本出ます。

各回で、各属性1本ずつに加えて、どれか1属性がもう1本光ります。つまり、毎回合計4本の塔が光ります。

直前に光った塔は、次の回では光らない想定です。

デバフは、氷耐性低下・雷耐性低下・火耐性低下が2人ずつ、無職が2人です。

### 無職

初期無職の人は、3回とも「2本光っている属性」の塔に入ります。

塔を踏むと属性耐性低下が追加で付きますが、初期無職の担当は変わりません。

### 属性耐性低下

属性耐性低下が付いた人は、自分の初期デバフ属性を基準に、時計回りに次の属性塔へ進んでいきます。

例：A側から時計回りに `氷 → 雷 → 炎` の順で塔が並んでいて、初期デバフが炎耐性低下の場合

- 1回目：氷塔
- 2回目：雷塔
- 3回目：炎塔

同じ初期デバフを持った2人で、同じ塔に入る想定です。

塔を踏むと、その塔の属性耐性低下が追加で付きます。

3回目の塔着弾直前に、無職以外の最初の耐性低下デバフは切れる想定です。

### カタストロフィ

1回目と3回目は、塔着弾と同時に2択のカタストロフィがあります。

- ケフカが風、緑：中安置
- ケフカが土、黄：外安置

回答表示では、踏んではいけない範囲を表示します。塔の中でも、表示された範囲を踏まない位置を選んでください。

緑のときは外側のドーナツ範囲、黄色のときは中央の円範囲を表示します。

## モード

### 位置確認

塔をクリックすると、その場で正解表示をします。

`一問ごと` のチェックが入っていない場合は、正解を一瞬表示して次のスライドへ進みます。

`一問ごと` のチェックが入っている場合は、正解表示後にその場で止まります。

### 実践練習

実践練習では、塔内の文字・数字と、光っている塔の白い点線を非表示にします。

緑の丸が自分の位置です。クリックした位置に移動できます。WASD、矢印キー、ゲームパッドでも動かせます。

タイムラインは以下の想定です。

- 初期デバフ付与：20秒
- 初期デバフ残り16秒：1回目カタストロフィの色表示・詠唱表示
- 10秒後：1回目塔着弾
- その7秒後：2回目塔着弾
- その6秒後：3回目塔着弾

回答表示は各1秒、3回終了後の結果表示は3秒です。

全体の流れを崩さないよう、シークバー側は回答表示分を差し引いて進行します。

`一問ごと` のチェックが入っていない場合は、3回終了後に結果を表示してから自動で次の問題へ進みます。

`一問ごと` のチェックが入っている場合は、結果表示後にその場で待機します。

実践中でも一時停止中でも、マップ右上の `＜` / `＞` で前後のスライドへ送れます。

## 表示オプション

### 詳細表示

チェックなしでは、パーティーリストのデバフをアイコンで表示します。

チェックありでは、`氷耐性低下` などの文字表示に切り替わります。

### マーカー

`2341 / 1234` ボタンで、フィールドマーカー表記を切り替えられます。

## 操作

- 塔クリック：位置確認で回答
- マップクリック：実践練習で自分の位置を移動
- WASD / 矢印キー：実践練習で移動
- ゲームパッド左スティック / 十字キー：実践練習で移動
- `＜` / `＞`：スライド送り
- `新しい問題`：別パターンを生成
- `正解表示`：現在スライドの正解を表示
- `一時停止 / 再開`：実践練習の進行を止める / 再開する

## 練習ログについて

この練習アプリでは、アプリ改善のため、練習ログを送信する場合があります。

送信されるのは、アプリ名、モード、問題種別、正誤、回答時間など、練習内容に関する情報のみで、個人情報は送信されません。同じブラウザ内での練習傾向を見るため、localStorageにランダムなブラウザIDを保存します。

練習ログにはIPアドレスを含めず、アプリ側でIPアドレスを保存しません。ログ送信はいつでもOFFにできます。
